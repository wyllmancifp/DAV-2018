Funciones, definición y uso

Una función es una porción de código reusable que puede ser llamada en cualquier momento desde nuestro programa. Se compone de una cabecera definitoria con un identificador y los parámetros a usar entre paréntesis. Seguido de un bloque de código a ejecutar cada vez que se llame a dicha función.

function nombre_funcion(parametros) {
instrucciones;
}
