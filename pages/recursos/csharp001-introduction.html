¿Qué es C#?

Un programa es un trozo de texto (código) escrito por un programador, formando un conjunto de instrucciones que fuerza directamente al ordenador a cumplir su cometido.

C# es un lenguaje de programación orientado a objetos desarrollado y estandarizado por Microsoft como parte de su plataforma .NET, que después fue aprobado como un estándar por la ECMA (ECMA-334) e ISO (ISO/IEC 23270). C# es uno de los lenguajes de programación diseñados para la infraestructura de lenguaje común.

Aunque C# forma parte de la plataforma .NET, ésta es una API, mientras que C# es un lenguaje de programación independiente diseñado para generar programas sobre dicha plataforma. Ya existe un compilador implementado que provee el marco Mono - DotGNU, el cual genera programas para distintas plataformas como Windows Microsoft, Unix, Android, iOS, Windows Phone, Mac OS y GNU/Linux.

Sintaxis y conceptos básicos

Su sintaxis básica deriva de C/C++ y utiliza el modelo de objetos de la plataforma .NET, similar al de Java, aunque incluye mejoras derivadas de otros lenguajes.


Los nombres de variables (lo que se conoce como "identificadores") pueden estar formados por letras, números o el símbolo de subrayado (_) y deben comenzar por letra o subrayado. No deben tener espacios intermedios. También hay que recordar que las vocales acentuadas y la eñe son problemáticas, porque no son letras "estándar" en todos los idiomas, así que no se pueden utilizar como parte de un identificador en la mayoría de lenguajes de programación.

Tampoco podremos usar como identificadores las palabras reservadas de C#. Por ejemplo, la palabra "int" se refiere a que cierta variable guardará un número entero, así que esa palabra "int" no la podremos usar tampoco como nombre de variable (pero no vamos a incluir ahora una lista de palabras reservadas de C#, ya nos iremos encontrando con ellas).

Hay que recordar que en C# las mayúsculas y minúsculas se consideran diferentes

En C# se usa una función "Main()" como punto de entrada de ejecución.

Las llaves { y } se usan para delimitar un bloque de programa. En nuestro caso, se trata del bloque principal del programa (Main).

Cada orden o sentencia de C# debe terminar con un punto y coma (;)

Para realizar comentarios en el código se utilizan los simbolos // para una sola línea y /* comentario */ para los comentarios multilínea

Variables y tipos de datos base

Una variable se utiliza para almacenar y hacer referencia a otro valor en el espacio de memoria del programa.

Para usar una cierta variable primero hay que declararla: indicar su nombre y el tipo de datos que querremos guardar. Dicha declaración se realiza escribiendo primero la palabra reservada del tipo de dato seguida de su identificador. Se les asigna un valor concreto, escribiendo su identificador el símbolo = y el valor a guardar.

Los tipos de datos numéricos más usados son:
- Entero con signo
sbyte: 8 bits, de -128 a 127
short: 16 bits, de -32,768 a 32,767
int: 32 bits, de -2,147,483,648 a 2,147,483,647
long: 64 bits, de -9,223,372,036,854,775,808 a 9,223,372,036,854,775,807

- Entero sin signo
byte: 8 bits, de 0 a 255
ushort: 16 bits, de 0 a 65,535
uint: 32 bits, de 0 a 4,294,967,295
ulong: 64 bits, de 0 a 18,446,744,073,709,551,615

- Punto flotante
float: 32 bits, de 1.5 × 10−45 a 3.4 × 1038, precisión de 7 dígitos
double: 64 bits, de 5.0 × 10−324 a 1.7 × 10308, precisión de 15 dígitos

- Decimal
decimal: 128 bits, al menos de –7.9 × 10−28 a 7.9 × 1028, con una precisión mínima de 28 dígitos

- Otros tipos de datos base son:
bool(Booleano): 0, 1 o true, false
enum(Enumeraciones)
struct(Estructuras)
char(caracteres Unicode)
string(cadenas de caracteres Unicode)

Operadores y estructuras de control

Operadores:
+ Suma
- Resta, negación
* Multiplicación
/ División
% Resto de la división ("módulo")

El orden de prioridad de los mismos será:
- En primer lugar se realizarán las operaciones indicadas entre paréntesis.
- Luego la negación.
- Después las multiplicaciones, divisiones y el resto de la división.
- Finalmente, las sumas y las restas.
- En caso de tener igual prioridad, se analizan de izquierda a derecha.

Otros operadores:
Unario
+x: identidad.
-x: negación.
!x: negación lógica.
~x: negación bit a bit.
++x: preincremento.
--x: predecremento.

Shift
x << y: desplazamiento a la izquierda.
x >> y: desplazamiento a la derecha.

Comprobación de tipos y relacional
x < y: menor que.
x > y: mayor que.
x <= y: menor o igual que.
x >= y: mayor o igual que.

Igualdad
x == y: igual que.
x != y: no igual que.

AND condicional
x && y: evalúa y solo si x no es false.
OR condicional
x || y: evalúa y solo si x no es true.

Funciones, definición y uso

Una función es una porción de código reusable que puede ser llamada en cualquier momento desde nuestro programa. Se compone de una cabecera definitoria con un identificador y los parámetros a usar entre paréntesis. Seguido de un bloque de código a ejecutar cada vez que se llame a dicha función.

function nombre_funcion(parametros) {
instrucciones;
}
