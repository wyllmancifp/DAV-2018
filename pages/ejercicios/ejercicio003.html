El circuito y el tiempo

Ejercicio 3 (Individual)

SpeedRun v:0.1.0


Enuncidado

Crear un objeto de juego vacío para el control del tiempo y se le añadirá el script crono.cs proporcionado.

En el canvas destinado a la interfaz de usuario "in-game" se añadirá un panel de texto con espacio suficiente para mostrar el tiempo del crono actual y los tiempos capturados por cada paso por "checkpoints".

Crear los modelos del arco a usar como checkpoint de la carrera. Se crearan como hijos de un "gameObject" vacío. Junto a los modelos crearemos tres objetos de juego más con un componente de unity "BoxCollider" cada uno. Dichos bloques de colisión han de cubrir la zona interna del modelo del arco. Se añadiran los scripts de control para los colisionadores y los tiempos proprocionados.

Configurar los atributos vacíos de los scripts usados y crear un prefab del objeto de juego del checkpoint creado anteriormente. Usando dicho prefab, posicionar, al menos, dos checkpoints más configurando el primero como "start", otro/s como "check" y el último como "end".

Adaptar el terreno de juego para que el usuario no pueda salirse del circuto o la zona de juego. Haciendo uso de colisionadores, alturas de terreno, ...
